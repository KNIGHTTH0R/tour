
*** УСТАНОВКА ****

1. Архив yii.zip необходимо распаковать в корневую директорию вашего веб-сервера (в результате в корне веб-сервера должна появиться директория yii).
2. Архив tour.zip необходимо распаковать в директорию домена, для которого устанавливается приложение (например, директория home/www/site.com).
3. Каталогам приложения assets и protected/runtime необходимо установить права на зпись (757).
4. В файле index.php приложения, необходимо изменить строку require(dirname(__FILE__) . '/../../yii/yii.php');.
   В строке необходимо указать путь к файлу yii.php (расположен в директории yii).
5. Создайте базу данных.
6. Импортируйте в базу данных данные из файла tour.sql (расположен в кроневой директории приложения).
7. В файле config/production.php в секции
		'db' => array(
			'connectionString' => 'mysql:host=localhost;dbname=tour',
			'username' => 'root',
			'password' => '',
			'charset' => 'utf8',
    необходимо указать настройки подключения к БД.
	
Если все шаги установки выполнены правильно, приложение должно запуститься. В качестве главной страницы должна отобразиться форма авторизации.
По умолчанию база данных уже содержит некоторые данные. Для входа в приложение вы можете использовать один из следующих демонстрационных профилей:
    логин: superadmin, пароль: superadmin - суперпользователь. Имеет доступ ко всем разделам без ограничений. Может добавлять роли и назначать права пользователям.
	
	логин: demo, пароль: demo - тестовый пользователь. Может работать с заявками и справочником клиентов.
	
Доступ к какому-либо разделу или действию в приложении определяется правами, которыми обладает пользователь, поэтому, прежде чем начать использовать приложение, необходимо настроить права доступа и роли приложения. После этого можно добавлять пользователей и назначать им права (роли).
